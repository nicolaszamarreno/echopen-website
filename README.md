EchOpen
=======

## Getting Started
**Install dependencies** to launch the project

 ```
 $ npm install
 ```

 Then, **launch the server & compilation** with this command

 ```
 $ npm start
 ```

  Then, **launch the server & compilation** with this command

 ```
 $ npm build:browserSync
 ```
